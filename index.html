Habilidades

HTML

HTML não é considerada uma linguagem de programação, mas sim uma linguagem de marcação.
É como se fosse a carcaça da página e nela colocamos os textos, links, imagens etc

CSS

CSS também é uma linguagem de marcação, e nela podemos estilizar nossa página.
Como uma maquiagem para a página, nela podemos escolher o layout dos textos, a fonte, tamanho das letras, cores de fundo e muito mais.
Existem diversas possibilidades de se alterar uma página.

JavaScript

Ao contrário das outras, o JavaScript é uma linguagem de programação, e nela acontece a parte não visual da página, chamada de Backend.
As funções de botões e interações do usuário com a página acontece nela.
